## Engi-Calc: Multi-Format Scientific & Engineering Calculator

This is a simple, menu-driven scientific and engineering calculator implemented in Python.

### Features

- **Matrix Algebra (NumPy)**: Addition, subtraction, multiplication, determinant, inverse, transpose.
- **Statistical Analysis**: Mean, median, and sample standard deviation for a list of numbers.
- **Unit Conversions**: Length, weight/mass, and temperature conversions.
- **Error Handling**: Uses `try-except` blocks to catch invalid input and division-by-zero–like errors (e.g., singular matrices).

### Requirements

- Python 3.9+ recommended
- NumPy

Install dependencies with:

```bash
pip install -r requirements.txt
```

### Running the Application

From the project directory:

```bash
python main.py
```
